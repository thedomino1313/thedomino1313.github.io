How to make a shortcut to your DS folder in WSL

  • Open WSL
  • Type vim ~/.bashrc
  • Welcome to vim
  • Use the arrow keys to navigate to the bottom of the file
  • Hit i to enter edit mode
  • Hit enter a couple of times to give yourself some room
  • Copy the following lines and use ctrl v or ctrl shift v to paste them into the file
if [ -f ~/.bash_aliases ]; then
    . ~/.bash_aliases
fi
  • Hit esc
  • Type :wq to save the file
    • This will take you back to the terminal
  • Type ~/.bash_aliases
    • This opens the file where you will put your aliases
  • Hit i to enter insert mode again
  • Optional: Add the following lines. The first allows you to easily get back to this file, and the second allows you to force update your changes to it.
alias val="v ~/.bash_aliases"
alias sal="source ~/.bash_aliases"
  • Add a shortcut to your Data Structures folder
alias ds="cd /mnt/C/Users/<the rest of the path>"
  • Where <rest of the path> is however you get to your Data Structures folder from the Users directory
  • Hit esc
  • Type :wq to save the file
  • Back in the terminal, type sal and hit enter.
    • If you didn’t add the sal alias from above, this will not work, and please type source ~/.bash_aliases instead.
